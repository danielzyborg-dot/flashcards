Card 1: What is a system?
Question
In your own words, what is a system of linear equations?
Try this first
Say what you’re trying to find (a point) and what must be true (both equations).
Think: two lines on the same coordinate plane.
A system of linear equations is two (or more) linear equations that share the same variables. The solution is the ordered pair that makes all equations true (the intersection point on a graph).
Card 2: What is a solution?
Question
What does the solution to a system represent on a graph?
Try this first
Picture two lines. Where do you look for the answer?
It’s the point both lines share.
The solution is the intersection point—the ordered pair that lies on both lines.
Card 3: Check a point (both equations)
System + Check
System:
y = x − 3
y = −1

Is (2, −1) a solution?
Try this first
Substitute x=2 into equation 1. Then check equation 2.
Does y become −1 in equation 1?
Equation 1: y = 2 − 3 = −1 ✓
Equation 2: y = −1 ✓

✅ Yes, (2, −1) makes both equations true.
Card 4: Check a point (only one works)
System + Check
System:
y = 2x − 2
y = x + 1

Does (3, 4) make both equations true?
Try this first
Plug (3,4) into each equation separately.
Compute 2(3)−2 and 3+1.
Eqn 1: y = 2(3) − 2 = 4 ✓
Eqn 2: y = 3 + 1 = 4 ✓

✅ Yes, (3,4) makes both equations true (it is a solution).
Card 5: Check a point (neither works)
System + Check
System:
y = −x + 1
y = 2x + 10

Is (3, −4) a solution?
Try this first
Substitute x=3 and compare with y=−4.
Eqn 1 gives y = −3 + 1 = −2.
Eqn 1: y = −3 + 1 = −2 (not −4) ✗
Eqn 2: y = 2(3)+10 = 16 (not −4) ✗

❌ No. It makes neither equation true.
Card 6: Solution types (one / none / infinite)
Question
A system can have one solution, no solution, or infinitely many solutions. Match each to the graph idea.
Try this first
Say what the lines do in each case.
Intersect once / never meet / same line.
One solution: lines intersect at 1 point.
No solution: lines are parallel (same slope, different intercepts).
Infinitely many: equations are the same line (same slope and same intercept).
Card 7: Decide the number of solutions (parallel)
System
System:
y = 2x + 3
y = 2x − 1

How many solutions?
Try this first
Compare slopes and y-intercepts.
Same slope, different intercepts.
Both have slope 2 but different y-intercepts (3 and −1) → parallel lines → ❌ No solution.
Card 8: Decide the number of solutions (same line)
System
System:
y = −3x + 4
y = −3x + 4

How many solutions?
Try this first
Are they identical?
Every point on one is on the other.
They are the same equation → the same line → ♾ infinitely many solutions.
Card 9: Graph meaning (intersection point)
Graph Fact
Two lines intersect at (−1, 2). What is the solution to the system?
Try this first
The solution is the shared point.
Just restate the intersection.
✅ The solution is (−1, 2).
Card 10: Complete the statement (is / is not)
Statement
Complete: The point (0, 4) is ____ the solution to the system if it lies on both lines.
Try this first
What word makes the sentence true?
If it lies on both lines, it works for both equations.
The point (0, 4) is the solution (because it makes both equations true).
Card 11: Graphing technique (y = mx + b)
How-To
How do you graph y = mx + b quickly?
Try this first
Name the two pieces: intercept and slope.
Plot b first, then use m as rise/run.
1) Plot the y-intercept b at (0, b).
2) Use slope m (rise/run) to find a second point.
3) Draw the line with a ruler.
Card 12: Solve by graphing (easy integers)
System
Solve the system (graphing or thinking):
y = x + 1
y = −x + 5
Try this first
Set them equal: x+1 = −x+5.
2x = 4.
x + 1 = −x + 5 → 2x = 4 → x = 2 → y = 3.
✅ Solution: (2, 3).
Card 13: Solve by graphing (another)
System
Solve the system:
y = 2x − 1
y = −x + 5
Try this first
Set them equal: 2x−1 = −x+5.
3x = 6.
2x − 1 = −x + 5 → 3x = 6 → x = 2 → y = 3.
✅ Solution: (2, 3).
Card 14: FAST-style multiple choice (solution point)
Multiple Choice
System:
y = x + 2
y = −2x + 1

Which point is the solution?
A. (0,2)   B. (1,3)   C. (−1,1)   D. (1,−1)
Try this first
Try plugging each point into BOTH equations until one works.
Check C first—it often works nicely.
Check C (−1,1):
Eqn1: y = −1+2=1 ✓
Eqn2: y = −2(−1)+1=3 (not 1) ✗

Try A (0,2): Eqn1 gives 2 ✓, Eqn2 gives 1 ✗
Try B (1,3): Eqn1 gives 3 ✓, Eqn2 gives −1 ✗
Try D (1,−1): Eqn1 gives 3 ✗

None of these are solutions as written. (This is a “trap-check” card.) On a real test, one option should work—so you’d re-check copying the equations.
Card 15: The ‘trap-check’ skill
Test Skill
If none of the answer choices work, what should you do before panicking?
Try this first
Think: copying errors and sign mistakes.
Re-check negatives and substitution.
1) Re-read the equations (signs!).
2) Re-plug the point carefully.
3) Make sure you didn’t swap x and y.
4) If graph is given, trust the intersection point.
Card 16: Identify solution type from slopes
Quick Decide
Without graphing: If two lines have different slopes, the system has…?
Try this first
What must happen if slopes differ?
They cross exactly once.
✅ One solution (the lines intersect exactly once).
Card 17: Identify solution type from slopes/intercepts
Quick Decide
Without graphing: If two lines have the same slope but different y-intercepts, the system has…?
Try this first
Same steepness, different starting heights.
Parallel.
❌ No solution (parallel lines).
Card 18: Identify infinite solutions quickly
Quick Decide
Without graphing: If two equations are exactly the same (or simplify to the same), the system has…?
Try this first
Every point works.
Same line.
♾ Infinitely many solutions.

Card 20: Two-part (graph + solution blanks)
Two-Part

Try this first

Card 22: Decide the correct statement about the system
Multiple Choice
Lines: y = 3 − 1.5x and y = x + 3.
Which statement is true?
☐ Same slope → no solutions

☐ Different slopes → one solution
☐ Different slopes → no solutions

Slopes are −1.5 and 1 → different → ✅ one solution.
